    01 · 裝置連線
    尚未連接 —— 離線

    填 ESP32 序列埠印出的 IP 位址(不用加 http:// 或結尾斜線)。手機/電腦要跟 ESP32 連同一個 Wi-Fi。若這個網頁是用 https 開啟(例如 GitHub Pages),部分瀏覽器可能會擋下對 http 裝置的連線 —— 若「連接 ESP32」一直失敗,可以改用 http://localhost:8000 本機伺服器方式開啟這個網頁測試。
